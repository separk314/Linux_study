# 1강

## 리눅스를 배워야 하는 이유

오픈소스 = 소스공개 + 라이선스(ex. GPL)

오픈소스를 사용할 때 리눅스가 필요.

## 리눅스 설치 방법

1. Virtual box, VM Ware 등의 가상화 툴 설치
2. window 10의 Linux kernel: WSL(Window Subsystem for Linux) 사용
3. ******************************************************************직접 컴퓨터에 리눅스 설치******************************************************************

## 컴퓨터 구성

1. CPU: 속도 빠름. cache, register 포함 
2. SSD, HDD: 보조기억장치, 속도 느림
3. Memory
4. Mainboard
5. power: 전원 공급
